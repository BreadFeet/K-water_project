Explanation on each chart related to health index data
1. 지역별 건강지표 차트

1) Line Chart

선택한 지역의 건강관련요인 중 분석하고자 하는 6가지 항목(삶의 질 지수, 양호한 주관적 건강수준 인지율, 스트레스 인지율, 우울감 경험률, 주관적 구강건강이 나쁜 인구의 분율, 연간 보건기관 이용률)를 확인할 수 있도록 하였습니다. 6가지 항목 모두 조율이 아닌 2005년 추계인구로 연령표준화된 표준화율을 사용하였습니다.

삶의 질 지수, 스트레스 인지율, 우울감 경험률 항목은 분명한 병리학적 원인이 있는 것이 아니며 여러 원인에 의해 영향을 받을 수 있는 가능성이 열려있는 항목이기 때문에 선정하였습니다. 예를 들어, 일상 용수의 심미적 요소에 의해 삶의 질이 영향을 받을 수도 있고, 우울감의 경우 특정 물질의 농도와 연관성을 보여서 인과관계를 의심해 볼 수 있지 않을까 하는 생각으로 선정하게 되었습니다.
또한 주관적 구강건강이 나쁜 인구의 분율은 매일 양치하면서 구강 내 노출되는 물질에 의해 영향을 받는 부분이 있지 않을지 확인하고자 하였으며, 양호한 주관적 건강수준 인지율, 연간 보건기관 이용률은 수질과 건강의 전반적인 연관성을 확인하기 위해 선정하였습니다.

원본 데이터의 경우 각 시도별 합계 데이터와 시군구 데이터가 모두 있으나, 수질 데이터와 공통된 지역을 뽑아내기 위해서 시도별 합계 데이터만을 사용하였습니다.

데이터셋
질병관리청 지역사회 건강조사 - 지역사회 건강관련 요인 데이터베이스 2008년~2019년 12개년 자료
데이터 형태
xlsx 다운로드 파일
데이터 전처리
매년 다른 sheet에 작성되어 있어서 각각의 데이터프레임으로 필요한 컬럼을 추출한 후에 12개 데이터프레임을 병합
1) 결측치 처리: 2008년에 '연간 보건기관 이용률' 항목이 집계되지 않아서 생긴 결측치는, 수질 데이터와 동일하게 0으로 치환함
2) 이후 함께 분석할 수질 데이터와 비교하여 양쪽 중 한쪽에 '세종특별자치시' 데이터가 없는 연도에 대해서는 세종시 데이터를 포함하지 않음
분석 라이브러리
Pandas
시각화 도구
Highcharts




2) Map Chart

연도를 설정하면 모든 지역의 6가지 항목을 한번에 확인, 비교할 수 있도록 하고자 하였습니다. 이를 위해 카카오맵 API를 사용하였고, 각 지역의 위도/경도 정보는 웹스크래핑을 통해 얻었습니다. 건강지표 데이터에서 설정된 지역은 시도 단위인데 반해 마커를 표시할 위도/경도는 하나의 도시로 설정해야 하기 때문에, 각 도의 도청 소재지 또는 위도/경도 값을 찾을 수 있는 근교의 다른 도시에 마커를 설정하였습니다.

데이터셋

데이터 형태
xlsx 다운로드 파일, 웹문서 스크래핑, API

2. 연도별 변화 추이

연관성이 있을것이라고 생각한 항목간의 상관관계를 볼 수 있는 Bubble plot을 연도별로 그린 뒤, 움직이는 이미지로 만들어서 연도별 변화 추이를 확인할 수 있도록 하였습니다. 연관성을 확인한 항목으로는 스트레스 인지율 - 우울감 경험률, 주관적 건강수준 인지율 - 연간 보건기관 이용률, 스트레스 인지율 - 삶의 질 지수입니다.
Bubble plot에서 인구수를 반영하여 버블마커의 크기를 지정하기 위해서 추가적으로 인구통계 자료를 가져와서 병합하였습니다.

데이터셋

행정안전부 주민등록 인구통계 - 2009년~2018년 10개년 자료
